# deal-leyu-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不依赖特定后端服务，可直接通过浏览器访问。

## 项目简介

deal-leyu-journal 是一个静态页面归档仓库。仓库内保存的 HTML 文件均为独立页面，彼此之间无强关联，也不面向任何特定网站或域名。其目的仅在于提供一种结构化的方式，用于保存和共享这些页面内容。

## 目录说明

仓库根目录下包含若干 HTML 文件及可能的子目录。典型的目录结构如下：

```
deal-leyu-journal/
├── README.md
├── index.html
├── page-1.html
├── page-2.html
└── assets/
    ├── css/
    ├── js/
    └── images/
```

- `*.html`：各个独立的 HTML 页面文件。
- `assets/`：存放页面引用的静态资源（样式、脚本、图片等），按类型分目录管理。

实际目录结构可能随归档内容调整，请以仓库当前状态为准。

## 页面归档说明

- 每个 HTML 页面均为完整、自包含的静态网页，可由浏览器直接打开。
- 页面之间无依赖关系，但可能共享 `assets/` 目录下的通用资源。
- 归档页面可能来源于不同时期、不同主题，统一存放于此以便查阅和管理。
- 页面内容不保证时效性或准确性，仅供存档参考。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改 HTML 页面。
- 提交信息将简要说明变更内容，例如新增页面、修复资源链接等。
- 欢迎通过 Issues 提出关于页面归档的建议或报告问题，但不承诺处理所有反馈。
- 仓库不提供在线演示或托管服务，如需查看页面效果，请克隆或下载后本地打开。

## 许可

本仓库内容仅用于归档目的。如需引用或使用其中页面，请自行确认原始内容的授权情况。仓库本身不声明特定许可证，除非另有说明。
